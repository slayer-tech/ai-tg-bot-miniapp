# Content Planner — Product

## Register
product

## Users
Администраторы Telegram-каналов: SMM, авторы, малый бизнес. Работают с телефона внутри Telegram.

## Product Purpose
Планировать, редактировать и публиковать посты в канал без выхода из Telegram.

## Jobs
- Смотреть календарь запланированных и опубликованных постов
- Создавать и редактировать посты (текст, фото, кнопка, расписание)
- Публиковать сейчас или по расписанию (МСК)
- Настраивать канал, автопилот, лимиты

## Tone
Уверенный, конкретный, без воды. Русский UI. Без emoji в интерфейсе.

## Anti-references
Generic Bootstrap cards, purple AI gradients, emoji navigation, перегруженные формы.
